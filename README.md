# yamerun-legal.github.io

法的文書（プライバシーポリシー・利用規約）を GitHub Pages でホスティングするリポジトリです。

## URL

| ページ | URL |
|--------|-----|
| トップ | https://yamerun.github.io/ |
| プライバシーポリシー | https://yamerun.github.io/privacy-policy/ |
| 利用規約 | https://yamerun.github.io/terms-of-service/ |

## 構成

```
.
├── index.html
├── privacy-policy/
│   └── index.html
└── terms-of-service/
    └── index.html
```

## 公開手順

1. GitHub でリポジトリ名 `yamerun-legal.github.io` を作成
2. このディレクトリをプッシュ
3. リポジトリの Settings → Pages → Branch: main / フォルダ: / (root) に設定
