Frequently Asked Questions

Basic operations

How do I create a folder?

Tap the "+" button at the bottom right of the home screen to create a new folder. Enter a folder name and save. Folders are the units that group outlines together.

How do I add a node?

On the outline screen (the screen you see after opening a folder), tap the "+" button in the toolbar at the bottom. A new node is added right after the currently selected node. After entering a title, you can also press the "Return" key to add the next node in sequence.

How do I change a node's indent level?

Use the "←" and "→" buttons in the toolbar in Write mode. Tap a node to select it, then use the buttons. Increasing the indent makes the node a child of the previous node; decreasing it moves the node up to its grandparent's level. If you indent a node that has children, the children move along with it.

What's the difference between "Write" and "Organize" modes?

Use the segmented button at the top of the screen to switch modes.

Write mode is focused on text input. You can directly edit the title and note of each node.

Organize mode lets you freely reorder nodes by drag & drop. You can also change indent levels. Tapping a node keeps it selected and automatically switches back to Write mode.

How do I edit a node's body (long-form notes)?

When you select a node in Write mode, a pencil icon appears on the right edge. Tap that icon to open the body editor, where you can write long-form rich-text content. The icon appears blue when a body exists.

How do I delete a node?

In Write mode, select the node you want to delete and tap the trash icon in the toolbar. A confirmation dialog appears. If the node has children, the children will also be deleted, so please be careful.

How do I collapse child nodes?

In Write mode, tap the "▼" icon on the left edge of a node that has children. The child nodes are collapsed and hidden. Tap again to expand.

About your data

Where is my data stored?

Data you enter, including folders, nodes, and body text, is stored only on your device. None of this data is transmitted to our servers or any external service.

To improve the app, we measure anonymous usage statistics such as screen navigation and operation counts using Google Analytics for Firebase. No personally identifiable information or user input content (such as node titles or body text) is transmitted. For details, please see the Privacy Policy.

Tell me about usage analytics

To improve the service, we collect the following kinds of anonymous information via Google Analytics for Firebase:

  • Which screens are viewed (screen navigation)
  • Counts of key actions such as adding or deleting nodes
  • How often you switch between Write and Organize modes
  • Launch count and time in app (session information)
  • Device information such as model name and OS version

All of this information is anonymized and is not used to identify individuals. To completely stop measurement, please uninstall the app. For details, please see the Privacy Policy.

Can I transfer my data when I change devices?

The app itself does not provide cross-device data migration (such as cloud sync). Whether app data is transferred during iCloud backup or device migration depends on your OS and device settings. Please back up important data yourself.

About ads

What is Tracking Permission (ATT)?

On iOS, a system tracking permission dialog may appear so that the app can deliver ads tailored to your interests. Even if you do not allow tracking, the main features of the app remain usable, and non-personalized ads may be displayed. For details, please see the "Tracking" section of the Privacy Policy.

Troubleshooting

The app won't launch or crashes immediately

Please try restarting your device, updating iOS, and updating the app to the latest version from the App Store. If the problem persists, please contact us with your OS version and a description of what happens.

My text isn't being saved

Text is saved automatically. Saving is finalized when you tap another node or navigate away. In rare cases, such as immediately after force-quitting the app, the last input may not be reflected. If this happens, please contact us.
